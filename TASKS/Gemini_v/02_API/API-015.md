- When: Endpoint `POST /api/reports/pdf` 로 Fetch를 수행함
- Then: 이 명세서가 지정한 응답값(바이너리 Blob 혹은 JSON URL) 규격에 완벽히 상응하여 리턴하고 프론트엔드가 이를 바탕으로 다운로드 트리거를 켤 수 있다.

- Given: Body나 Query에 식별자가 없는 쓰레기값이 인입됨
- When: 호출 시도됨
- Then: HTTP Status 400 Bad Request 리턴을 선언한 명세대로 반환된다.
