- Given: 올바른 형식의 증상 설명 텍스트가 담긴 DTO
- When: `createAsTicket` 을 호출함
- Then: Zod 검사가 이를 에러 없이 스키마 객체로 파싱하여 반환한다.

- Then: `{ errors: { symptomDescription: "자세히 기입해주세요" } }` 포맷으로 클라이언트에 에러 구조체가 하달된다.
