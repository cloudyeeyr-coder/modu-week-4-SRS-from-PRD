### Phase A: 예약 상태 및 모델 선언
- [ ] `O2oBooking` 예약 모델 (Phase 1 에서는 스키마만 생성, 실기능은 연기됨)
  ```prisma
  model O2oBooking {
    id                String   @id @default(cuid())
    buyerCompanyId    String
    visitDate         DateTime @db.Date
    region            String
    assignedManagerId String?
    status            String   @default("requested") // requested, confirmed, completed, cancelled
    reportSubmittedAt DateTime?
    reportContent     Json?    
    createdAt         DateTime @default(now())
    updatedAt         DateTime @updatedAt

    buyerCompany BuyerCompany @relation(fields: [buyerCompanyId], references: [id])
  }
  ```
