### Phase A: 유니버설 알림 스키마 
- [ ] 수신자가 다중 역할(Buyer, SI 등 구분없이 UUID) 구조 맵핑.
  ```prisma
  model Notification {
    id          String   @id @default(cuid())
    receiverId  String   // 특정 Role 테이블 참조 없이 범용 계정 ID 활용 권장
    type        String   // INFO, WARNING, ACTION_REQUIRED
    content     String
    isRead      Boolean  @default(false)
    channel     String   // DB_ONLY, EMAIL_SENT, KAKAO_SENT 플래그
    createdAt   DateTime @default(now())
  }
  ```
