## :white_check_mark: Task Breakdown (실행 계획)

### Phase A: 보증 조건 및 릴레이션 구조 잡기
- [ ] `Contract`, `EscrowTx` 양쪽 테이블 모두에 대해 참조 확보
  ```prisma
  model Warranty {
    id                   String   @id @default(cuid())
    contractId           String   @unique // 혹은 1:N으로 가도 무방하나 보통 계약당 1건 처리
    escrowTxId           String
    asCompanyName        String
    asContactPhone       String
    asContactEmail       String?
    coverageScope        String   // 보증 범위 텍스트 요약
    coveragePeriodMonths Int      @default(12)
    issuedAt             DateTime @default(now())
    pdfUrl               String?  // 보증서 실물 바이너리 주소
    createdAt            DateTime @default(now())

    contract Contract @relation(fields: [contractId], references: [id])
    escrowTx EscrowTx @relation(fields: [escrowTxId], references: [id])
  }
  ```
